草地營位

自然舒適的草地空間

我們的草地營位位於露營區最平坦的區域，提供鬆軟舒適的草地，是帳篷露營的理想選擇。每個營位都有充足的空間，適合家庭或朋友小團體共享美好的露營時光。

營位特色：

  • 平整柔軟的草地，適合各種帳篷搭設
  • 每個營位約15坪空間，提供舒適的活動範圍
  • 配備營火區和野餐桌
  • 附近有水電插座和照明設施
  • 步行可輕鬆抵達公共衛浴和廚房區
  • 自然樹蔭遮蔽，夏日也能保持涼爽
價格資訊：

平日：NT$ 1,200 / 帳 / 晚
週末及國定假日：NT$ 1,500 / 帳 / 晚
(最多可容納4位成人，額外每人加收NT$ 300)

木棧板營位

平整乾燥的木棧板營位

木棧板營位是我們最受歡迎的選擇之一，特別是在雨季。平整的木板地面確保您的帳篷保持乾燥，不受潮濕地面的影響，讓露營體驗更加舒適。每個木棧板區域都經過精心設計，提供絕佳的視野。

營位特色：

  • 高架木棧板設計，不怕地面潮濕
  • 每個營位約12坪空間，附帳篷固定點
  • 配備獨立營火區和堅固的木製野餐桌
  • 每個營位配有專屬電源插座
  • 部分營位可遠眺山景或河景
  • 鄰近公共烤肉區，方便準備美味餐點
價格資訊：

平日：NT$ 1,500 / 帳 / 晚
週末及國定假日：NT$ 1,800 / 帳 / 晚
(最多可容納4位成人，額外每人加收NT$ 300)

露營車營位

全配備的露營車空間

為露營車和露營拖車設計的專屬區域，提供完整的水電接口和排水設施。這些營位位於露營區較為平坦的外圍，方便進出，同時保有私密性和寧靜感。

營位特色：

  • 平整的碎石地面，適合露營車和拖車停靠
  • 提供30A電力接口、自來水接口和排水設施
  • 每個營位約20坪空間，含車位和活動區
  • 配備遮陽棚、固定式烤肉爐和野餐桌
  • 便利的垃圾收集點和回收設施
  • 鄰近共用洗衣設施
價格資訊：

平日：NT$ 2,000 / 晚
週末及國定假日：NT$ 2,300 / 晚
(最多可容納6位成人，額外每人加收NT$ 300)
